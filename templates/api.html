API Endpoints

POST /api/predict

Analyze an email for phishing content

Request Body

{
  "email_text": "Email content to analyze"
}

Response

{
  "prediction": "Phishing",
  "probability": 87.65,
  "threat_level": "High",
  "action": "Block",
  "indicators": [
    {
      "name": "Suspicious Phrases",
      "description": "Contains 3 suspicious phrases",
      "severity": "high"
    },
    ...
  ],
  "suspicious_count": 5
}

Example Usage

curl -X POST https://your-app-url/api/predict \
     -H "Content-Type: application/json" \
     -d '{"email_text": "Dear customer, your account needs to be verified. Click here: http://bit.ly/suspicious-link"}'

Response Fields

Field Type Description
prediction string "Phishing" or "Legitimate"
probability number Confidence percentage (0-100)
threat_level string "High", "Medium", or "Low"
action string Recommended action to take
indicators array List of detected risk indicators
suspicious_count number Count of suspicious elements found

Rate Limits & Usage

Please observe these guidelines when using our API:

  • Maximum 100 requests per hour
  • Maximum email size: 1 MB
  • Please include your application name in User-Agent header
  • For higher limits, please contact us
